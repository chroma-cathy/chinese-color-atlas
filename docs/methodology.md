# Methodology

Chinese Color Atlas treats Chinese traditional colors as both cultural references and practical design assets.

## Reliability Tiers

- `verified`: color names with direct classical-text or source support.
- `attributed`: useful modern or compiled names without direct classical attestation.
- `disputed`: names or interpretations with public scholarly objection.
- `qianlong measured`: colors linked to published Qianlong Palette CIE Lab measurement research.

## Design Workflow Exports

The project turns traditional colors into formats designers and developers can use immediately:

- CSS custom properties
- Tailwind theme tokens
- W3C DTCG-style tokens
- Figma Variables JSON
- Procreate swatches
- Adobe ASE swatches
- AI prompt samples

This public repository contains free samples and token examples. The complete packs are available at https://chinesecoloratlas.com/resources.
